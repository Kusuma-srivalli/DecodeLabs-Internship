# Recommendation System - Project 3

## Objective
To build a simple AI-based recommendation system that suggests items based on user preferences.

## Project Description
This project takes user input (interests) and matches them with a dataset of items using simple logic and similarity scoring. It then recommends the most relevant items to the user.

## Features
- Takes user input (comma separated interests)
- Matches preferences using similarity logic
- Calculates score for each item
- Displays ranked recommendations
- Shows results in order of best match

## Technologies Used
- Python

## How to Run
1. Open terminal
2. Run the file using:
   python recommendation.py

## Example Input
space, sci-fi, action

## Example Output
Interstellar (match score: 2)  
The Matrix (match score: 2)  
Avengers (match score: 1)
