# Data Classification Using Artificial Intelligence

## Project Overview
This project demonstrates a basic machine learning classification model using the Iris dataset.

## Objective
To train a machine learning model that can classify iris flowers into different categories based on their features.

## Tools and Technologies
- Python
- VS Code
- Scikit-learn
- Pandas
- NumPy

## Dataset
Iris Dataset (built-in dataset from Scikit-learn)

## Algorithm Used
Decision Tree Classifier

## Steps Performed
1. Loaded the Iris dataset.
2. Split the dataset into training and testing sets.
3. Trained the Decision Tree model.
4. Predicted results on test data.
5. Evaluated model accuracy.

## Result
The model successfully classified iris flowers with good accuracy.

## Project Structure
- ai_classification.py → main code file
- screenshots/ → output images
- requirements.txt → dependencies

## How to Run
1. Clone the repository
2. Open terminal in project folder
3. Install dependencies:
   pip install -r requirements.txt
4. Run the program:
   python ai_classification.py

## Requirements
- Python 3.x
- pandas
- numpy
- scikit-learn
