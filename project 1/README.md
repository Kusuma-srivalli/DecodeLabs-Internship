# Rule-Based Chatbot Project

## Objective
To create a simple rule-based chatbot that responds to predefined user inputs using Python.

## Project Description
This chatbot interacts with users in a continuous loop and gives responses using simple if-else logic. It can handle greetings and exit commands.

## Key Features
- Handles greetings (hello, hi)
- Responds to basic user inputs
- Handles exit commands (bye, exit)
- Runs in a continuous loop

## Technologies Used
- Python

## How to Run
1. Open terminal
2. Run:
   python chatbot.py

## Example
You: hello  
Chatbot: Hello! How can I help you?

You: bye  
Chatbot: Goodbye! Have a nice day.
