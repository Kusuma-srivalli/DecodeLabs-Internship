# Data Classification Using Artificial Intelligence

## Project Overview
This project demonstrates a basic machine learning classification model using the Iris dataset.

## Objective
To train a machine learning model that can classify iris flowers into different categories based on their features.

## Tools and Technologies
- Python
- VS Code
- Scikit-learn

## Dataset
Iris Dataset

## Algorithm Used
Decision Tree Classifier

## Steps Performed
1. Loaded the Iris dataset.
2. Split the dataset into training and testing sets.
3. Trained the Decision Tree model.
4. Predicted results on test data.
5. Evaluated model accuracy.

## Result
The model successfully classified iris flowers with high accuracy.
